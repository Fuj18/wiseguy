Documentation for Wiseguy
=========================

This tool helps system administrators integrate web applications into
their web service infrastructure.  It supports application deployment
by providing a uniform configuration mechanism to construct WSGI
application stacks.  This tool has several aspects:

 * a configuration schema for WSGI components
 * a catalog of 3rd party WSGI components
 * a configuration format for WSGI application stacks
 * an interface to build WSGI stack configurations
 * a mechanism to build a WSGI application stack instance
